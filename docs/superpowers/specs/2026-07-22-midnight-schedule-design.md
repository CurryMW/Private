# 钉钉日报凌晨 00:30 定时设计

## 目标

将钉钉 AI 日报的唯一自动推送时间从北京时间 08:30 调整为每天北京时间凌晨 00:30。

## 工作流

GitHub Actions cron 使用 UTC。北京时间凌晨 00:30 对应 UTC 前一日 16:30，因此 `.github/workflows/daily.yml` 只保留：

```yaml
schedule:
  - cron: "30 16 * * *"
```

手动预演、手动正式运行、每日一次状态保护、模型故障通知和内容兜底逻辑均保持不变。GitHub 定时任务仍可能因平台负载延迟，不能保证在 00:30 整秒到达。

## 验收

- 工作流测试精确断言唯一 cron 为 `30 16 * * *`。
- 中文 README 中所有自动推送时间统一为北京时间 00:30、UTC 16:30。
- 完整离线测试通过。
- 推送远程后 GitHub Test 工作流成功。
- 不自动触发真实钉钉消息。
